# SOC SIEM Detection Lab

This repository contains hands-on Security Information and Event Management (SIEM) labs focused on log analysis, authentication monitoring, threat detection, and security operations workflows.

The objective of these labs is to develop practical experience using a SIEM platform to collect, search, analyze, and investigate security-related events.

## Objectives

* Learn SIEM fundamentals and log ingestion.
* Analyze Windows authentication activity.
* Investigate failed logon events.
* Monitor PowerShell activity.
* Create basic detection logic.
* Build dashboards used in security operations.

## Labs

### Lab 01 — Splunk Setup and Log Ingestion
Configure Splunk and ingest Windows security logs.

### Lab 02 — Authentication Event Analysis
Review successful and failed authentication events.

### Lab 03 — Failed Logon Detection
Investigate failed authentication activity and identify suspicious patterns.

### Lab 04 — PowerShell Activity Monitoring
Review PowerShell-related events and administrative activity.

### Lab 05 — Detection Rule Creation
Create basic detections for suspicious activity.

### Lab 06 — Security Dashboard Creation
Build dashboards for monitoring authentication and security events.

## Disclaimer

These labs were performed in a personal lab environment for educational and portfolio development purposes.
